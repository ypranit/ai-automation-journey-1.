# 🤖 AI Automation Journey #1

A multi-modal Telegram AI Assistant built with Python and deployed on Railway.

## Features

- 💬 AI Chat using OpenRouter
- 🎤 Voice Message Transcription using Whisper
- 📰 News Search
- ☁️ Railway Deployment
- 🔄 GitHub Auto Deploy
- 👤 User Profiles
- 📝 Persistent Notes

## Tech Stack

- Python
- Telegram Bot API
- OpenRouter
- OpenAI Whisper
- Railway
- GitHub

## Commands

/start - Start bot

Send:
- Text messages
- Voice notes
- News queries
